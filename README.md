
# AgroCADI PRO v3

Protótipo funcional de sistema para análise de solo.

## Instalação

Instale Python 3.10+

## Dependências

pip install -r backend/requirements.txt

## Rodar servidor

cd backend
uvicorn main:app --reload

Abra navegador:

http://localhost:8000/docs

## Interface

Abra o arquivo:

frontend/index.html
